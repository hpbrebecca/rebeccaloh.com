# Company Logos

Bitte fügen Sie die folgenden Logo-Dateien in diesen Ordner ein:

1. **finma.svg** - FINMA Logo
2. **iba-ag.svg** - IBA AG Logo  
3. **gemeinde-koeniz.svg** - Gemeinde Köniz Logo
4. **ficas-ag.svg** - FICAS AG Logo

Die Logos können im SVG-Format oder als PNG-Dateien (mit transparentem Hintergrund) gespeichert werden.

Falls die Logos nicht geladen werden können, werden automatisch Text-Fallbacks angezeigt.
